# webtech

### Installs node modules
```
npm install
```

### Compiles and minifies frontend pages
```
npm run build
```

### Startss the server
* #### localhost https
```
node server.js
```
* #### cloud https
```
node server.js -cloud
```
* #### http only
```
node server.js -http
```
* #### query the database
```
node server.js -sql
```
